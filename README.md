## Innoverse36

#Initializing the project

1. Clone the repository
2. Run `cd Server`
3. Run `npm install`
4. Run `cd client`
5. Run `npm install`
6. Run `npm start` - for server (backend)
7. Run `npm run dev` - for client (frontend)
